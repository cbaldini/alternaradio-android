## 🚀 Cómo Instalar

### Opción 1: ADB (Línea de comandos)
```bash
adb uninstall ar.alternaradio.app
adb install -r app/build/outputs/apk/release/app-release.apk
```

### Opción 2: Copiar a Desktop y enviar por WhatsApp
```bash
cp app/build/outputs/apk/release/app-release.apk ~/Desktop/alterna-radio-v3.apk
```

Luego:
1. Abre Finder → Desktop
2. Arrastra `alterna-radio-v3.apk` a WhatsApp
3. En el Motorola, abre el archivo y tapa instalar

### Opción 3: Si pide permisos en el Motorola

Si aún aparece advertencia de Google Play Protect:
1. Toca "INSTALAR DE TODAS FORMAS"
2. O ve a Configuración → Aplicaciones → Instalar desde fuentes desconocidas
